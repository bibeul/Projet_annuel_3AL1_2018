## Usage
### Command lines to build the java client
```bash
git clone https://github.com/bibeul/Projet_annuel_3AL1_2018.git
cd Projet_annuel_3AL1_2018
cd client_lourd
mvn clean install
java -jar client_lourd.jar
```

### Command lines to build the node api
```bash
git clone https://github.com/bibeul/Projet_annuel_3AL1_2018.git
cd Projet_annuel_3AL1_2018
cd api_PA
npm start
```
